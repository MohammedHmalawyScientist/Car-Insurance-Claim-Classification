# Project No. (1) : Car Insurance Claim Classification

### Description:
Now, companies can find out the real customer behaviors over the data.
This is a web app to predict whether a customer will claim the insurance or not.

### Used Features:
- Age
- Gender
- Driving Experience
- Income
- Credit Score
- Vehicle Ownership
- Vehicle Year
- Married
- Children
- Annual Mileage
- Vehicle Type
- Speeding Violations
- Duis
- Past Accidents
- Education
- Postal Code
